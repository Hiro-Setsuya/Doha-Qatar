# Doha, Qatar — Tourism Website (Activity 5 — Web Development)

This repository contains Activity 5 for a Web Development course/exercise. It is a responsive tourism website showcasing the beauty, culture, and highlights of Doha, Qatar, demonstrating HTML, CSS, and JavaScript fundamentals with modern web design principles.

## What this repo contains

This repository is a simple front-end project. It's intended for learning, showcasing responsive design, and demonstrating interactive UI components including carousels, animations, and mobile-friendly navigation.

### File structure

Root files

- `index.html` — Landing page with full-screen video background.
- `home.html` — Main content page with highlights, architecture carousel, and landmarks.
- `README.md` — This file.

Assets

- `assets/css/style.css` — Main CSS stylesheet with responsive design and animations.
- `assets/img/` — Images of Doha landmarks, architecture, and attractions.
- `assets/script/script.js` — Client-side JavaScript for scroll animations and interactivity.
- `assets/vid/` — Video content for the landing page.

Directory tree (visual):

```
index.html
home.html
README.md
assets/
  css/
    style.css
  img/
    (Doha images and landmarks...)
  script/
    script.js
  vid/
    (video files...)
```

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations, Media Queries)
- JavaScript (Intersection Observer API)
- Bootstrap 5.3.2
- Google Fonts (Playfair Display, Poppins)
